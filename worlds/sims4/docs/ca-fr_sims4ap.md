## Qu'est-ce que la randomization fait au jeu?

Les compétences ne peuvent pas être augmentés au-delà du nombre d'items que vous avez reçu pour chaque compétence.
Les futurs versions pourrait inclure une randomization pour les exigences en matière de compétences professionelles. 

## Quel est le but des Sims 4 lorsqu'elle est randomisée?

Couramment, le but est de compléter une aspiration choisie avec un Sim.

## Quels items peuvent être dans le monde d'un autre joueur?

Les compétences sont des items de progressions, les Simoleons sont des items de poubelle.

## À quoi ressemble un item d'un autre monde dans les Sims 4?

Lorsque vous atteignez un **check**, un message sera affiché dans le jeu.

## Quand le joueur reçoit un item, qu'est-ce qui se passe?

Un message avec l'item sera affiché, maintenant le joueur peut améliorer ses compétences dans le compétence spécifiée
